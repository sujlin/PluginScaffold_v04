# Hotfix: XML-first plugin output path

This package writes MA3 plugin XML/Lua files to:

```text
~/MALightingTechnology/gma3_library/datapools/plugins/<pluginSubdir>/
```

This is the user plugin library used by MA3 Import/Library workflows. Do not import old `*.plugin.xml` sidecar files generated by earlier scaffold versions. Use `DeskLock.xml` or build with `--xml-mode embedded` for a single XML installer.


# 热修复：XML 优先插件输出路径

此软件包将 MA3 插件 XML/Lua 文件写入以下位置：

```text

~/MALightingTechnology/gma3_library/datapools/plugins/<pluginSubdir>/

```

这是 MA3 导入/库工作流程使用的用户插件库。请勿导入早期脚手架版本生成的旧版 `*.plugin.xml` 边车文件。请使用 `DeskLock.xml` 或使用 `--xml-mode embedded` 构建单个 XML 安装程序。
